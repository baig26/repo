Movie ticket booking

The project is a web-based application created using JSP, HTML, and Bootstrap. It allows administrators to manage theaters and users to book movie tickets. The user-friendly interface enables customers to select their desired movie, showtime, and seat. The system also provides real-time updates on available seats, seat selection, and payment options. The admin panel allows theater managers to add, modify, or delete movie listings, manage the seating capacity, and track user activity. The project is designed to streamline the movie ticket booking process for users and provide an efficient and user-friendly experience for theater administrators.

Online Course Registration

A student registration system was developed with an easy-to-use online form, allowing students to register themselves without the need for multiple manual forms. The user-friendly design was intended to save time and money, as filling out the form online was more efficient compared to the laborious process of filling out several manual forms. The system aimed to simplify the registration process for students, making it accessible and streamlined.

Emotion detection using real time video

Emotion recognition systems aim to apply emotional knowledge to enhance human-computer communication and improve user experience. By enabling computers to sense users' emotional states and react accordingly, communication can become more satisfying. These systems can also be developed for various medical applications, such as aggression detection, stress detection, and the detection of conditions like autistic disorder, Asperger's syndrome, hepatolenticular degeneration, and frustration.
